# Proyecto del reloj

¡Bienvenido/a! Este proyecto se trata de un reloj que tiene varias funciones que ire exlicando. El reloj sera realizado con los siguientes lenguajes de programación: **javascript, html y css.

Este proyecto tendra varias funciones. Ahora voy a explicar las siguientes funciones.

# 1º La función de ver la hora actual.

Esta función se hará con un campo para poder introducir con js la hora exacta que es en este mismo momento.

# 2º La función de poder poner una alarma.

La función de poder poner una alarma con la hora y minutos que quiera la persona que lo use. Y también podra escoger los dias que quiera activar la alarma. Esta función se podrá hacer con los input de number para que puedan introducir las numeros necesarios, este programa sera de 24 horas.
Y la función para escoger los dias se podra escoger con un select que permita escoger varias opciones.
